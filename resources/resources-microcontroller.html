Brief introduction about arduino

Arduino is basically a specially designed hardware that is solely made to create projects without using analog components. In simple words, it is a mini-brain which takes inputs, processes them and gives output. All these are electric pulses. All you have to do is write code, and the brain is ready to do that work. It takes power to form a battery. You have to write a code and upload it, and the code will be in your own tiny digital brain called arduino.





Why you should know about it and use it

When you actually start working on your project without an arduino, you have to deal with all the analog components like resistors, capacitors etc.. All these are actually very uncertain in nature and blew up. When you are about to complete your project, if anything goes wrong with the hardware, it means you have lost everything you cannot check for each component separately , here where arduino comes where you do not have to deal with this analog and just concentrate on your project and its features. Thus it becomes way easier because you don’t have to start from scratch and the platform is already built for you.

How to Download Arduino IDE





Step 1:

You can get different versions of Arduino IDE from the Download page on the Arduino Official website. You must select your software, which is compatible with your operating system (Windows, IOS, or Linux). After your file download is complete, unzip the file.





Step 2:

After your Arduino IDE software is downloaded, you need to unzip the folder. Inside the folder, you can find the application icon with an infinity label (application.exe). Double-click the icon to start the IDE.





Step 3:

Once the software starts, you have two options −

  • Create a new project.
  • Open an existing project example.

To create a new project, select File → New. To open an existing project example, select File → Example → Basics → Blink.





Step 4:

To avoid any error while uploading your program to the board, you must select the correct Arduino board name, which matches with the board connected to your computer. Go to Tools → Board and select your board.





Step 5:

Select the serial device of the Arduino board. Go to Tools → Serial Port menu. This is likely to be COM3 or higher (COM1 and COM2 are usually reserved for hardware serial ports). To find out, you can disconnect your Arduino board and re-open the menu, the entry that disappears should be of the Arduino board. Reconnect the board and select that serial port.

Components of Arduino







Power port

An AC-to-DC adapter or a battery could be used to power the Arduino board. The power source can be connected by plugging a 2.1 mm positive centre plug into the power socket of the board. The recommended voltage for most Arduino models is between 6 and 12 Volts.





USB connector

This is a Type B USB port used to load a program from the Arduino IDE onto the Arduino board. The board can also be powered through this port.





Power Indicator LED

This LED should light up whenever you plug your Arduino into a power source. If this light doesn’t turn on, there’s a good chance something is wrong.





Reset Switch

When this switch is pressed, it sends a logical pulse to the Microcontroller's reset pin, restarting the programme from the beginning. If your code doesn't repeat but you want to test it several times, this is a great option.





GND PIN

GND Is An Abbreviation Of Ground. There are 3 GND pins on the Arduino, any of which can be used to ground your circuit.





Power Input Pins (3.3v and 5v)

This Pin Is Used To Supply Power To The Entire Circuit

  • 3.3V − Supply 3.3 output volt
  • 5V − Supply 5 output volt





Power Input (Vin)

This pin can also be used to feed the Arduino board from an external power source, such as AC power.





Digital pins:

You can find Digital pins labelled “Digital 0 to 13.” These pins can be used as either input or output pins. When used as output, these pins act as a power supply source for the components connected to them. When used as input pins, they read the signals from the component connected to them. When digital pins are used as output pins, they supply 40 milliamps of current at 5 volts. Some of the digital pins are labelled with a tilde (~) symbol next to the pin numbers (pin numbers 3, 5, 6, 9, 10, and 11). These pins act as normal digital pins but can also be used for Pulse-Width Modulation (PWM), which simulates analog output like fading an LED in and out.





Analog input pins:

The Arduino UNO board has 6 analog input pins, labelled “Analog 0 to 5.” These pins can read the signal from an analog sensor like a temperature sensor and convert it into a digital value so that the system understands. Although these pins are labelled analog and are analog input by default, these pins can also be used for digital input or output.





Pin 13 LED

Arduino Uno features an integrated LED connected with digital pin 13. When the pin is HIGH, the LED illuminates and when it is LOW the LED is disabled.





TX – RX LEDs:

TX means transmission and RX means reception. These are indicator LEDs that flash when the UNO board sends or receives data.





RX – TX Pins

These are serial communication pins, used to communicate with other Arduino boards as well as computers.





Coding in Arduino UNO:





Programing of Arduino is done on Arduino IDE which is written in functions of C and C++.So basically to code on arduino we need to know C & C++





FUNCTIONS:

In arduino there are many defined functions





Input/Output syntax:

In arduino there are two ways of doing the connections, first by using the digital pins and secondly by using the analog pins.



For the digital pins functions used for input/output are:

  1. digitalRead(): it reads the values specified for a specific digital pin
  2. digitalWrite(): it can write a HIGH/LOW value to a digital pin
  3. pinMode(): it Configures the specified digital pin to behave either as an input or an output



For the analog pins functions used for input/output are:

  1. analogRead() : Reads the value of specified analog pin.
  2. analogReference(): configures the reference voltage used for analog input
  3. analogWrite(): Writes a analog value to the pin



There are other advanced input/output methods such as noTone() ,pulseIn() ,pulseInLong() ,shiftIn() ,shiftOut() ,tone() etc







Other syntaxes:

Syntax to assign random variables:

We can also assign random numbers using random() ,randomSeed()



Math functions:

Maths function such as abs(),min(),max(),pow(),sq(),sqrt() are allowed on arduino IDE and have their usual meaning



Time delay:

Giving delay to the program is not an issue as there are two ways to put a program delay {delay(),delayMicroseconds()}



Data types:

Data Types such as arrays,size_t, boolean, string, unsigned int etc are used in Arduino IDE. Interconversion of these data types is also allowed on Arduino for eg: we can use int() to convert a data type to integers



STRUCTURES:

These elements are based on Arduino(C++) code. Arduino IDE provides various control structures such as goto, switch...case,for loop ,if...else statement ,while loop,break,continue,return etc. It also provides all the different types of operators such as arithmetic, boolean, bitwise, comparison etc.
