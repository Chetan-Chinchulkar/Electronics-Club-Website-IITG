Brief introduction about arduino

Arduino is basically a specially designed hardware that is solely made to create projects without using analog components. In simple words, it is a mini-brain which takes inputs, processes them and gives output. All these are electric pulses. All you have to do is write code, and the brain is ready to do that work. It takes power to form a battery. You have to write a code and upload it, and the code will be in your own tiny digital brain called arduino.





Why you should know about it and use it

When you actually start working on your project without an arduino, you have to deal with all the analog components like resistors, capacitors etc.. All these are actually very uncertain in nature and blew up. When you are about to complete your project, if anything goes wrong with the hardware, it means you have lost everything you cannot check for each component separately , here where arduino comes where you do not have to deal with this analog and just concentrate on your project and its features. Thus it becomes way easier because you don’t have to start from scratch and the platform is already built for you.

How to Download Arduino IDE





Step 1:

You can get different versions of Arduino IDE from the Download page on the Arduino Official website. You must select your software, which is compatible with your operating system (Windows, IOS, or Linux). After your file download is complete, unzip the file.





Step 2:

After your Arduino IDE software is downloaded, you need to unzip the folder. Inside the folder, you can find the application icon with an infinity label (application.exe). Double-click the icon to start the IDE.





Step 3:

Once the software starts, you have two options −

  • Create a new project.
  • Open an existing project example.

To create a new project, select File → New. To open an existing project example, select File → Example → Basics → Blink.





Step 4:

To avoid any error while uploading your program to the board, you must select the correct Arduino board name, which matches with the board connected to your computer. Go to Tools → Board and select your board.





Step 5:

Select the serial device of the Arduino board. Go to Tools → Serial Port menu. This is likely to be COM3 or higher (COM1 and COM2 are usually reserved for hardware serial ports). To find out, you can disconnect your Arduino board and re-open the menu, the entry that disappears should be of the Arduino board. Reconnect the board and select that serial port.

Circuit Design Details:

VR = variable resistances, i.e. potentiometers Both the circuits are built using LM386 IC (Audio amplifier chip). The laser is our means of communication i.e. the laser will transmit the signals from transmitter to the receiver. The laser we are going to use is a laser diode (5V DC 5mW laser dot diode). Then a solar panel will receive the signal from laser light and give it as input to the receiver circuit. This is the starting point of the receiver circuit. The final output of the receiver part is the original audio that we sent from mobile which is played on the speaker. The transmitter circuit will receive the audio signal from mobile via an aux cable. The aux cable can be bought separately or can be obtained from an earphone. The signal from aux cable is sent into the circuitry shown in diagram built around the LM386.

The first potentiometer the signal encounters is used to control the input impedance the LM386. That will control how much of input signal actually goes into the amplifier. The capacitor and potentiometer between terminals 1 and 8 will be used to control the gain of the audio amplifier. The voltage divider network formed by R2, R3 and VR3 keeps the voltage as well as the current for the laser diode in the safe region. That will do it for the transmitter part.

Now we send the signal through the diode and receive it via the solar panel. This signal is fed to input pin 3 of IC through coupling capacitor C5. There is some processing to be done before it can be output from the speaker. The main tasks to be done are amplification and removal of carrier frequencies from the input. The LM386 IC will do the amplification. The gain of the amplifier is fixed by capacitor C7. Its output will be passed through a low pass filter circuit formed by C8, R4, C9 and LS1 to remove carrier frequencies. VR4 is used to change the signal level from the solar panel. The output is ready to be played by a speaker.



Noise Removal:


Think that the project is over?? Not yet. A major headache will be the noise coming from the speaker. Some common sources of noise in the circuit are-

  • Channel Noise
  • Cheap speakers
  • Unwanted signals recieved by the solar panel due to the surrounding lights
  • Noise due to circuit

Noise due to LM386 IC and channel was significantly reduced by employing various noise reduction techniques :

  • Filter feedback using a capacitor and a resistor acts as an Anti- Hiss Feedback network. This can be used as an effective measure that eliminates the hissing noise and gives clearer audio signal.
  • Ground loop voltage was reduced by keeping all the ground terminals as close as possible.
  • Adding a capacitor between the battery terminals.
